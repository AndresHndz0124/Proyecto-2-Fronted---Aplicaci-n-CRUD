CRUD

A partir del siguiente formulario pondremos en pratica un concepto denominado CRUD que nos permite manipular datos, listar información y modificación de la misma dentro de la interfaz de usuario.

Operaciones basicas dentro de CRUD:

  • C - Create ("Crear")
  • R - Read ("Leer")
  • U - Update ("Actualizar")
  • D - Delete ("Borrar")

Un ejemplo muy común relacionado con operaciones CRUD son las tablas de datos. En el siguiente formulario se basa en un "Ejemplo" de almacenamiento de información de todos los roles que pueden conformar el area de una diseño de producto de una empresa.

Para poder hacer uso de este formulario debes ingresar datos correspondientes al Rol, descricpción de las funciones y el salario aproximado.
